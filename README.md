# Pruebas Unitarias - Repositorio de Entrega

Este repositorio contiene las prácticas realizadas en la unidad didáctica de **Pruebas Unitarias de Software**. El objetivo es validar funciones lógicas mediante scripts automatizados con PHPUnit, siguiendo estándares de calidad de código.

## Estructura del Proyecto

El repositorio está organizado por semanas de clase:

* **`/s7`**: Práctica inicial de configuración de entorno y pruebas básicas.
* **`/s8`**: Implementación de lógica booleana (esPar, esPositivo, etc.) y configuración de `phpunit.xml`.

## Tecnologías Utilizadas

* **Lenguaje**: PHP 8.x
* **Framework de pruebas**: PHPUnit 11
* **Gestor de dependencias**: Composer

## Instrucciones de Uso

Para ejecutar las pruebas en tu entorno local:

1. Asegúrate de tener instalado Composer.
2. Navega a la carpeta de la semana correspondiente (ej. `cd s8`).
3. Instala las dependencias:
   ```bash
   composer install
